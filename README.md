# Professional Portfolio Website

## Objective

The objective of this project is to design and develop a responsive personal portfolio website using HTML5, CSS3, and JavaScript. The website presents professional information, technical skills, projects, and contact details in a clean and user-friendly format.

## Tech Stack

* HTML5
* CSS3
* JavaScript

## Features Implemented

* Multi-page website structure
* Responsive navigation bar
* Home page with introduction
* About page with profile summary
* Skills section
* Projects showcase section
* Contact page with form
* Responsive design for desktop, tablet, and mobile devices
* Dark and Light theme toggle
* Smooth scrolling
* Clean and organized folder structure

## Setup Instructions

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open index.html using Live Server.
4. Explore all pages through the navigation menu.
